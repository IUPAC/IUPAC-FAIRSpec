# DRYAD PRE-PROCESSING SCRIPT

This script is used to:

1. Fetch Dryad dataset from the Dryad website (make sure to add `./env` file in the `dryad_script` folder)

2. Generate some *prediction* files on the Dryad website based on Emma's algorithm (IUPAC-FAIRSPEC/stolaf/emma)

- `compound_candidate_identifier.py` is updated on 01/29/2026

- Run the script: `bash dryad_preprocessing.sh`
